# Анализ культурных трендов

Система для анализа тональности и тематического моделирования пользовательских комментариев на платформах культурных событий.

## Функции
- Сбор комментариев с Яндекс.Афиши и Tripster
- Предобработка текста 
- Классификация тональности (negative/neutral/positive) — Accuracy 88.8%
- Тематическое моделирование (BERTopic) — выделение 40+ тем

## Результаты модели
Модель классификации тональности показала следующие результаты:

- Accuracy:  0.888
- Macro F1:  0.891
- Negative:  F1 = 0.861
- Neutral:   F1 = 0.859
- Positive:  F1 = 0.952
